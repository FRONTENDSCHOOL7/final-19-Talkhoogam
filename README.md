- <톡후감>은 서로의 독후감을 공유하는 **독서 SNS 플랫폼**입니다.
- 책과 함께 **짧은 톡후감 또는 긴 독후감 모두 공유**할 수 있습니다.
- 중고책을 거래할 수 있는 **거래 서비스**를 제공합니다.
- 마음 맞는 사람끼리 모여 독서할 수 있는 **독서 모임** 구인 서비스를 제공합니다.
